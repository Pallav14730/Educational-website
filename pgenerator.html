What is python Generator

Python Generators are the functions that returns the tranversal object and used to create iterators. It tansverse the entire items at once. The Generator can also be an expression in which syntax is similar to the list comprehension in python.

There is a lot of complexity in creating iterators in python; we need to implement __iter__() and __next__() method to keep track of internel ststes.

It is a length process to create iterators. That's why the Generator plays an essential role in simplifying this process. If there is no value found in iteration,It raises stopiterators execption.

How to create Generator function in python

It is quite to create a generator in python. It is similar to the normal funciton defined by the def keyword and uses a yield keyword instead of return. or we can say that if the body of any function contains a yield statement,it automatically becomes a generator function. consider the folloiwng example:

def simple():
for i in range(10):
if(i%2==0):
yield i

for i in simple()
print(i)

yield vs. return

The yield statement is responsible for controlling the flow of the generator function. It pauses the function execution by saving all states and yielded to the caller. Later it resumes execution when a successive function is called. We can use the multiple yield statement in the generator function.

The return statement returns a value and terminates the whole funciton and only one return statement can be used in hte function.

Difference between Generator function and normal funciton

1. Normal function contains only one Lreturns statement whereas generator. function can contains one or more yield statement.

2. When the generator function are called, the normal function is paused immediately and control transferred to the caller.

3. Local Variable and their states are rembered between successive calls.

4. Stopilteration exception is raised automatically when the function terminates.
