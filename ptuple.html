Python Tuple

Python Tuple is used to store the sequence python object. The tuple is similar to list since the value of the items store in the list can be changed,whereas the tuple is immutable, and the values of the items stored in the list can be changed.

Creating a tuple

A Tuple can be written as the collection of commas-seperated(,) values enclosed with the small () brackets. The parantheses are optional but it is good practice to use. A tuple can be defined as follows:

 T1={101,"peter",22}
T2={"Apple","Banana","Orange"}
T3=10,20,30,40,50

Tuple indexing and slicing
The indexing and slicing in the tuple are similar to list. The indexing in the tuple starts from 0 and goes to length(tuple)-1.

The item in the tuple can be accessed by using the index[] operator. python also allow us to use the colon operator to access multiple item in the tuple.

Consider the following image to understand the indexing and slicing in detial.

 tup={1,2,3,4,5,6,7}
print(tup[0])
print(tup[1])
print(tup[2])

print(tup[8])

Negative Indexing
The tuple element can also access by using negative indexing. The index of -1 denotes the rightmost element and -2 to the second last item and so on.

The element from left to right are transversed using the negative indexing. Consider the following example.

 tuple1={1,2,3,4,5}
print(tuple1[-1])
print(tuple1[-3:-1])
print(tuple1[:-1])
print(tuple1[-2:])
 Output :
5
2
(3,4)
(1,2,3,4)
(4,5)
 Deleting Tuple
Unlike list, the tuple items cannot be deleted by using the del keyword as tuples are immutable. To delete an entire tuple, we can use the del keyword with tuple name.

consider the following examples.

 tuple={1,2,3,4,5,6}
print(tuple1)
del(tuple1)
print(tuple1)
del tuple1
print(tuple)
 (1,2,3,4,5,6)
