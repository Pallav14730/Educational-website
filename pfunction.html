Python Function

Function are the most important aspect of an application.A function can be defined as the organized block reusable code, which can be called whenever required.


Python allows us to divide a large program into the basic building blocks known as a function.The function contains the set of programming statement enclosed by {}. A function can be called multiple times to provide reusability and modularity to the python program.

The function helps to programmer to break the program into the smaller. It organizes the code every effictively and avoid repetition of the code. As the program grows, function makes the program more organized.

Pythom allows us various inbuilt function like range() or print(). Although, the user can create its function, which can be called user-defined function.

1. User-defined function- The user-defined function are those define by the user to perform the specific task.

2. Bulit-in fuunction- The built-in function are those function that are pre-defined in python.

In this tutorial, we will discuss the user define function.

Advantage of Function in python

There are the following advantage of function.

1. Using function,we can avoid rewriting the same logic/code again and again and anywhere in program.

2. We can call python function multiple times in a program and anywhere in a program.

3. We can track a large python program easily when it is divided into multiple function.

4. Reusability is the main achievement of python function

5. However, Function calling is always overhead in a python program.

Function calling

In python, after the function is created, we can call it from another function. A function must be defined before the function call; othrewise, the python interpreter gives an error. To call the function, use the function name followed by the parenthesis.

Consider the following example of a simple example that prints the message "Hello World".

hello_world

The return statement

The return statement is used at the end of the function and returns the result of the function. It terminates the function execution and transfers the result where the function is called. The return statement cannot be used outside of the function.

It can contain the expression which gets evaluated and value is returned to the caller function. if the return statement has no expression or does not exist itself in the function then it returns the None object.

Default Arguments
Python allows us to initialize the argument at the function definition.If the value of any of the argument is not provided at the time of function call, then that argument can be initialize with the value given in the definition even if the argument is not specified at the function call.

 Keyword argument
Python allows us to call the function with the keyword argument. This kind of function call will enables us to pass the argument in the random order.

The name of the argument is treated as the keywords and matched in the function calling and definition,If the same match is found, the values of the argument are copied in the function definition.

The variable defined outside any function is known to have a global scope,whereas the variable defined inside a function is known to have a local scope.
