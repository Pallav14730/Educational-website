Python String

Till now, we have discussed numbers as the standard data-types python. In this section of the tutorial, we will discuss the most popular data type in python, i.e,string.

Python string is the collection of the characters surrounded by single quotes, double quotes, or triple quotes.The omputer does not understand the characters; internally, it stores manipulated character as the combination of the 0's and 1's.

Each character is encoded in the ASII or unicode character.So we can say that python string are also called the collection of unicode character.

In python,string can be created by enclosing the character or the sequence of character in the quotes.Python allows us to use single quotes,double quotes,or triple quotes to create the string.

 str 1='Hello python'

print(str2)

Reassigning String

Updating the content of the string is as easy as assingning it to a string.The string object doesn't support item assigment i.e, A string cna only be replaced with a new string since its content cannot be partially placed.String are immutable in python.

 str="Hello"

str[0]="h"

print(str)
 Output :

str[0]="h"
