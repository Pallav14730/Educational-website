Python Decorator

Decorator are one of the most helpfull and powerful tools of python. These are used to modify the behaviour of the function. Decorators provides the flexibility to wrap another function to expand the working of wrapped function, without permentatly modifying it.

"In Decorators,function are passed as an arguments into the another function and called inside the wrapper function."

It is also called meta programming where part of the program attempts to change another part of program at compile time .

What are the function in python ?

Python has the most interesting feature that everything is treated asn an object even classes or any variable we define in python is also assumed as an object. Function are First class object in the python because they can reference to ,passed to a variable and returned from other function as well. The example is given below.

def func1(msg):
print("Hi")
func2=func1
func2("Hi")
Output :

Hii
Hiu
In the above program, when we run the code it give the same output for both function. The func2 refered to function func1 and act as function. We need to understand the following concept of the function.

1. The function can be referenced and passed to a variable and returned from other funcion as well.

2. The function can be declared inside another function and passed as an arguments to another function.
