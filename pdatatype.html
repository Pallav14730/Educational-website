Python Data type

Variables can hold values, and every value has a datatype. Python is a dynamically typed language; hence we donot need to define the type the of the variable while declaring it. The interpreter implicity binds the value with its type.


a= 5


The variables a holds integer value five and we didnot define its type. Python interpreter will automatically interpret variables a as an integer type.

Python enables us to check the type of the variables used in the program.Python provides us the type() funtion, which returns the type of the variable passed.

consider the follwing example to define the values of different data type and checking its type.

 a=10
b="Hi python"
c= 10.5
print(type(a))
print(type(b))
print(type(c))

 Output:

< type 'int'>
< type 'str'>
< type 'float'>

Numbers

Numbers stores numeric values.The integer, float,and complex values belongs to a python numbers datatype. Python provides the type () function to known the datatype of the variable. Similarly, the isinstance() function is used to check an object belongs to a particular class.

python creates numbers object when a number is assigned to a variable.for example.

1. int- Integer value can be any length such as integers 10,2,29,-20,-150 etc. python has no restriction on the length of an integer.Its value belongs to int

2.Float- Float is used to store floating-point numbers like 1.9,9.202,15.2,etc. It is accurate upto 15 decimal point.

3. Complex- A complex number contains an ordered pair, i.e, x+iy where x and y denote the real and imaginery parts, respectively. The complex numbers like 2.14j,2.0+ 2.3j, etc

String

The string can be defined as the sequence of character represented in the quotation marks. In python, we can use single,double, or triple quotes to define a string.

String handling in python is a straightforward task since python provides built-in function and operation to perform operation in the string.

In the case of string handling, the operator +used to conceterate two strings as the operation. "hello"+"python" return " hello python".
