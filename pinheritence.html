Python Inheritence

Inheritence is an important aspect of the object-oriented paradigm. Inheritence provides code reusability to the program because we can use an existing class to create a new class instead of crreating it from scratch.

In inheritence, the child class acquires the properties and can accesss all the data members and function defined in the parent class.
