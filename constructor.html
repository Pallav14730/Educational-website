Constructor

In Java , a constructor is a block of codes similar to the method. It is called when an instance of the class is created. At the time of calling constructor, memory for the object is allocated in the memory.

It is a special type of method which is used to initialize the object.

Every time an object is created using the new() keyword, at least one constructors is called.

It calls a default constructor if there is no constructor available in the class.In such case , java compiler provides a deafult constructors by default.

There are two types of constructors in Java: no-arg-constructor, and parameteried constructor.

Rules for creating java constructor

There are two rules defind for the constructor.

1. constructors name must be the same as its class name.

2. A constructors must have no explicit return type.

3. A java constructors cannot be abstract, static, final and synchronized.

Types of Java constructors

1. Deafult constructor(no-arg-constructors)

2. parameterized constructor

Java Default Constructor

A constructor is called "Default constructors" when It doesn't have any parameter.

 syntax of default constructor:
