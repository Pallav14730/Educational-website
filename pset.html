Python Set

A python set is the collection of the unordered items. Each element in the set must be unique, immutablem, and the sets remove the duplicate elements. Sets are mutable which means we can modify it after its creation.

Unlike other collection in python, there is no index. However, we cannot directly access any element of the sets by the index. However,we can print them all together, or we get the list of elements by looping through the set.

Creating a Sets

The sets can be created by enclosing the commas-seperated immutable items with the curly braces{}. python also provides the set() method , which can be used to create the set by the passed sequence.

Despite the fact that discard() and remove() method both perform te same tast, There is one main difference between discard() and remove().
