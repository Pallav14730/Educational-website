Java Switch Statement

The java switch statement executes one statement from multiple condition. It is like if-else ladder statement. The switch statement works with byte, short,int, long,enum types,string and some wrapper types like short int, and long.Since java 7,you can use strings in the statement.

In other words, the switch statement tests the equality of a variable against multiple values.
 Points to Remember 1. There can be one or N number of case values for a switch expression.

2. The case value must be of switch expression type only. The case value must be literal or constant. It doesn't allow variable.

3. The case values must be unique.In case of duplicate value,it render compiler-time error.

4.Each case statements can have a break statements which optional. when control reaches to the break statement,it jumps the control after the swtich expression. If a break statement is not found, it executes the net case.

5. The case value can have a default label which is optional.
 Syntax: Switch(expresson)
{
case value1;
break;
case value2;
break;
......
default:
// code to be executes if all cases are not match;
}
 Example: public class SwitchExample
{
public static void main(string[] args)
{
int number=20;
swtich(number)
{
case10:system.out.println("10");
break;
case20:system.out.println("20");
break;
case30:system.out.println("30");
break;
deafult:system.out.println("Not in 10,20,30");
}
}
}
 Output: 20
