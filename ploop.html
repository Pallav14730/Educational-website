Python Loop

The flow of the program written in any programming language is sequential by default. Sometimes we may need to alter the flow of the program. The execution of a specific code may need to be repeated several numbers of times.

for this purpose, the programming language provides various types of loop which are capable of repeating some specific code several number of times. Consider the following diagram to undertand the working of a loop statement.

Why we use loops in Python ?

The looping simplifies the complex problems into the easy ones. It enables us to alter the flow of the program so that instead of writing the same code again and again, we can repeat the same code for a finite number of times. For example,if we need to print the first 10 natural numbers then, instead if using the print statement 10 times, we can print inside loop which runs up to 10 ilteration.

Advantage of loops

There are the following advantage of loops in python.

1. It provides code re-usability

2. Using loops,we donot need to write the same code again and again.

3. Using loops, we can travel over the element of data structure.
