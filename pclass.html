Python Class And Object

We have already discussed in previous tutorial, a class is avirtual entity and can be seen as a blueprint of an object.The class into existance when it intantiated. Let's understand it by am example.

Suppose a class is a prototype of a building.A building contains all the details about the floor, rooms, doors, window,etc. We can make as many building as we want, based on these details. Hence, the building can be seen as a class, and we can create as many object of this class.

On the other hand, the object is the instance of class.The process of creating an object can be called instantiation.

In this section of the tutorial, we will discuss creating classes and object in python.We will also discuss how a class attribute is accessed by using the object.
