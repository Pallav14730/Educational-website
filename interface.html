Interface in Java

An interface in Java is a bluefrint of a class. It has a static constant and abstract methods.

The interface in java is a mechnism to achieve abstraction. There can be only abstract methods in the Java interface, not methods body. It is used to achieve abstrction and multiple inheritence in Java.

1. Java interface also interface also represents the IS-A reltionship.

2. It cannot br instantiated just like the abstract class.

How to declare an interface

An interface is declared by using the interface keyword. It provides total abstraction; means all the methods in an interface are declared with the empty body, and all the fields are public , static and final by default . A class that implements an interface must implements all the methods declared in the interface.

 Syntax: interface {
//declare constant fields
// declare methods that abstract
// by default
}

Java Interface Example

interface Bank{
float rateofinterest();
}
class PNB implements Bank{
public float rateofinterest(){
return 9.7f;
}
class SBI implements Bank{
public float rateofinterest(){
return 9.15f;
}
class Testinterface2{
public static void main(string[] args){
Bank b=new SBI();
system.out .println("ROI:"+b.rateof interest());
}
}
 Output: ROI: 9.15
