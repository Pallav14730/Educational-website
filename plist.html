Python List

A List in python is used to store the sequence of various types of data. Python list are mutable type its mean we can modify its element after it created. However, python consist of six data-types that are capable to store the sequence, but the most common and reliable type is the list.

A list can be defined as a collection of values or items of different types. The items in the list are seperated with the commas(,) and enclosed with the square brackets[].

A list can be define as below

L1=["john",102,"usa"]
L2=[1,2,3,4,5,6] Output :

print(type(L1))
print(type(L2))

Characters of List

The list has the following Characteristics


1. The list are ordered .

2. The element of the list can access by index.

3. THe list are the mutable type.

4. A list can store the number of various element.
