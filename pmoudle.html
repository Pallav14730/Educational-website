Python Moudle

A Python mudule can be defined as a python file which contains a python code including python function,class or variable. In other words,we can say that our python code file saved with the extension (.py) is treated as the moudle.We may have a runnable code inside the python moudle.

Moudle in python provides us the flexibility to organize the code in a logical way.

To use the functionality of one moudle into another,we must have to import the specific moudle.

Example

In this Example,we will create a moudle named as file.py which contains a function func that contains a code to print some message on the console.

Let's create the moudle named as file.py

def displayMsg(name)
print("Hi"+ name);

Loading the Module in our python code

we need to load the moudle in our python code to use its functionality.Python provides two types of statement as defined below.

1. The important statement

2. The from-import statement

The import statement

The import statement is ussed to import all the functionality of one moudle into another. Here,we must noticed that we can use the functionality of any python source file by importing that file as the moudle into another python source.

We can import multiple moudles with a single import statement, but a moudle is loaded once regardless of the number of time, it has been imported into our file.

Example
import file;
name=input("Enter the name ")
file.displayMsg(name)
Output:

Enter the name kartik
Hi kartik
The from-import statement
Instead of importing the whole moudle into the namespace, python provides the flexibility to import only the specific attributes of a moudle.This can be done by using from? import statement. The syntax to use the from-import statement is given below.

calculation.py: def summation(a,b):
return a+b
def multiplication(a,b):
return a*b;
def divide(a,b):
return a/b;
Main.py: from calculation import summation
a=int(input("Enter the first number"))
b=int(input("Enter the second number"))
print("sum= ",summation(a,b))
Output :

Enter the first number 10
Enter the second number 20
sum=30
