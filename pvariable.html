Python Variable

Variable is a name that is used to refer to memory locaiton.Python variable is also known as an identifier and used to hold the value.

In python, we don't need to specify the type of of variable because python is a infer language and smart enough to get variable type.

Variable names can be a group to both the letters and digits, but they have to begin with a letter or an underscore.

It is recommanded to use lowercase letters for the variable name, Rahul and rahul both are two different variable.

Identifer Naming

Variables are the example of identifier. An Identifer is used to identify the literals usesd in the program.The rules to name an identifier are given below.


1. The first characters of the variable must be an alphabets or underscore ( _ ).

2. All the characters except the first characters may be an alphabets of lower -case(a-z), uppercase(A-Z), underscore,or digit(0-9).

3. Identifier name must not contain any white-space, or special character (!,@,#,%, ^,&)

4. Identifer name must not be similar to any keyword defined in the language.

5. Identifer names are case sensitive; for example, my name,and MyName is not the Same.

Declaring Variable and Assigning Values

Python does not bind us to declare a variable before using it in application.It allow us to create a variable at the required time.

We don't need to declare explicity variable in python. When we assign any value to the variable,that variable is declared automatically.

The equal (=) operator is used to assign values to a variable.
