Data types

Data types specifies the different sizes and values that can be stored in the varaible. There are two types of datatype in java.

1. Primitive datatypes: The primitive datatype include boolean, char byte, short, int,long, float and double.

2. Non- primitive datatype: The non primitive datatypes include classes, interface and arrays.

Java primitive Datatypes

In java language,primitive datatypes are the building blocks of data manipulaiton.These are the most basic datatype available in Java language.

There are 8 types of primitive datatypes: 1. boolean datatype

2. byte datatype

3. char datatype

4. short datatype

5. int datatype

6. float datatype

7. double datatype

Boolean Data type

The Boolean datatype is used to store only two possible values: true or false. This datatype is used for simple flags that track true/flase conditions.

Example: Boolean one= false

Byte Datatype

The byte datatype is an example of primitive datatype. It is an 8-bit signed two's complement integer.Its value-range lies between -128 to 127 It minimum value is -128 and maximum value is 127. Its default value is 0.

the datatype is used to save memory in large arrays where the memory saving is most required. It saves space because a byte is 4 times smaller than an integer.It can also be used in place of "int" and data type.

Example: byte a=10, byte b=-20
Short Datatype
The short data type is a 16-bit signed two's complement integer. Its value-range lies between -32,768 to 32,767.Its default vlaue is 0.

The short datatype can also be used to save memory just like byte datatype .A Short datatype can also be used to save memory just like datatype.A short datatype is 2 times smaller than an interger.

Example : short s=10000, short r=-5000
Int Data Type
The int data type is a 32-bit signed two's complement integer.Its value-range lies between -2,147,483,648 to 2,147,483,647. Its default value is 0.

Example: int a=100000, int b=-200000 Long Datatype
The long datatype is a 64-bit two's complement integer. Its value range lies between -9,223,372,036,864,775,808 to 9,223,372,036,854,775,808.
Example : long a=1000000L, long b=-2000000L Float Datatype The float datatype is a single-precision 32 bit. Its value range is unlimited.It recommanded to use a float if you need to save memory in large arrays of floating point numbers.The float data type should never be used for precise values, such as currency.Its default value is 0.0f

Example: float f1=234.5f
Char Datatype
The char datatype is a single 16-bit unicode character. Its value range lies between '\u0000' to '\ufff'. The char datatype is used to store characters.

Example: char letterA='A'
