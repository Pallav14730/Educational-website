Python Dictionery

Python Dictionery is used to store the data in a key-point pair format. The dictionery is the data type in python, which can simulate the real-life data arrangment where some specific value exist for the some particular key. It is the mutable data-structure.The dictionery is defined into element keys and values.menu

1. Key must be a single element
2. vlaue can be any type such as list,tuple,integer,etc

In other words, we can say that a dictionery is the collection of key-vlaue pairs where the value can be any python object. In contrast, the key are the immutable python object, i.e, Numbers, string, or tuple.

Creating the Dictionery

The dictionery can be created by using multiple key-value pairs enclosed with the curly brackets{}, and each ey is seperated from its vlaue by the colon(:) The syntax to define the dictionery is given below.

 Let's see an example to create a dictionery and print its content.

 Dict={}
print("Empty Dictionery:")
print(Dict)


Dict =dict({1:'java',2:'T',3:'point'})
print("\n Create Dictionery by using dict():")
print(Dict)



Dict=dict([1, 'kartik'),(2, 'Sharma')])
print("\n Dictionery with each item as a pair:")
print(Dict)
 Output :

Empty Dictionery:
{}

Create Dictionery by using dict():
{1: 'Java', 2: 'T', 3: 'point'}

Dictionery with each item as a pair
{1: 'Kartik','Sharma'}

Accessing the dictionery values

We have discussed how the data can be accessed in the list and tuple by using the indexing.

However,the vlaues can be accessed in the dictionery by using the keys as keys are unique in the dictionery .

The dictionery value can be accessed in the following way.

Adding dictionery vlaues
The dictionery is a mutable datatype, and its values can be updated by using the specific keys. The values can be updated along with the key Dict[Key]=value. The update() method is also used to update an existing values.

Note: If the key-value already present in the dictionery, the value gets updates. Otherwie ,the new keys added in the dictionery.

Properties of Dictionery Keys
1. In the dictionery, we cannot store multiple values for the same keys. If we pass more than one value for a single key, then the vlaue which is last assigned is considered as the value of the key.

2. In python, the key cannot be any mutable object, we can use numbers, string, or tuples as the key,but we cannot use any mutable object like the list as the key in the dictionery.
