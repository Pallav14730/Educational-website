 C++ Crash Course Call by value and call by reference in C++
 There are two ways to pass value or data to function in C++ language:Call by value and call by reference. Original value is not modified in call by value but it is modified in the call by reference.

 In call by value, being passed to the function is locally store by the function parameter in stack memory location. If you changed the value of function parameter,it is changed for the current function only.It will not change the value of variable inside the caller method such as main().
