Python Constructor

A constructor is special type of method which is used to initilize the instance members of the class.

In C++ or java, the constructor has the same name as its class,but it treats constructor differently in python.It is used to create an object.


1. Parameterized Constructor

2. Non- Parameterized

Constructor definition is executes when we create the object of this class.Constructors also verify that there are enough resources for object to perform any start-up task.

Example :

class Employee:

def_init_(self,name,id):
self.id=id

self.name=name

def display(self):

print("ID: % \nName: %s" %(self.id,self.name))
emp1=Employee("john",101)

emp2=Employee("David",102)

emp1.display()
emp1.display()
 Output:

ID: 101
Name: john
ID: 102
Name: David
